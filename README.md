# SwiftSite Labs

A modern, responsive single-page website for SwiftSite Labs - a web development agency that builds websites for businesses.

## Features

- **Modern UI/UX**: Clean, minimal design with smooth animations
- **Fully Responsive**: Mobile-friendly across all device sizes
- **Contact Form**: Integrated with Google Sheets for form submissions
- **Projects Showcase**: Display of completed projects
- **Performance Optimized**: Fast loading with Vite build tool

## Tech Stack

- HTML5
- CSS3 (with custom properties and modern features)
- Vanilla JavaScript
- Vite (development server)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/swiftsite-labs.git
cd swiftsite-labs
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.
